# SisteMail — Pan Pa' Ya!

Sistema interno de comunicaciones corporativas desarrollado para **Pan Pa' Ya!**, Multipan y Multicarriel.

🌐 **Producción:** [sistemail-panpaya2.web.app](https://sistemail-panpaya2.web.app)

---

## Tecnologías

- **Frontend:** HTML5, CSS3, JavaScript ES6+
- **Base de datos:** Firebase Firestore (tiempo real)
- **Autenticación:** Firebase Auth
- **Almacenamiento:** Cloudinary
- **Hosting:** Firebase Hosting

## Funcionalidades

- 📧 Correos internos con hilo de respuestas y adjuntos múltiples
- 💬 Chat multimedia (texto, audio, imagen, video, documentos)
- 🗓️ Reuniones con solicitud, aprobación y recordatorios automáticos
- 🚨 Alertas críticas con notificaciones push y sonido
- 📊 Estadísticas y mapa de sucursales en tiempo real
- 👥 Control de acceso con roles: Usuario, Técnico, Admin, Superadmin
- 🏢 Soporte multi-empresa: Pan Pa' Ya!, Multipan, Multicarriel

## Configuración

1. Crea un proyecto en [Firebase Console](https://console.firebase.google.com)
2. Copia tu `firebaseConfig` y reemplázalo en `index.html`, `admin.html` y `usuario.html`
3. Configura una cuenta en [Cloudinary](https://cloudinary.com) y actualiza `CLOUD_NAME` y `PRESET` en los archivos JS
4. Despliega con `firebase deploy`
